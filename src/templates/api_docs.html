Gemini-CLI-Termux API 文档

📚 Gemini-CLI-Termux API 文档

认证信息：所有API端点（除特殊说明外）都需要Basic认证，用户名：admin，密码：GEMINI_AUTH_PASSWORD环境变量值（默认：123）
聊天补全
POST
/v1/chat/completions

提供与OpenAI兼容的聊天补全接口，支持流式输出。

请求示例：
curl -X POST http://localhost:8888/v1/chat/completions \\
-H "Content-Type: application/json" \\
-H "Authorization: Basic YWRtaW46MTIz" \\
-d '{
"model": "gemini-2.5-pro",
"messages": [
{"role": "user", "content": "Hello!"}
],
"temperature": 0.7,
"max_tokens": 1000
}'
模型列表
GET
/v1/models

获取支持的模型列表信息。

请求示例：
curl -X GET http://localhost:8888/v1/models \\
-H "Authorization: Basic YWRtaW46MTIz"
模型详情
GET
/v1beta/models

获取详细的Gemini模型信息。

内容生成
POST
/v1beta/models/{model}/generateContent

使用指定Gemini模型生成内容。

流式生成
POST
/v1beta/models/{model}/streamGenerateContent

流式生成内容。

健康检查
GET
/health

服务健康状态检查（无需认证）。

控制面板
GET
/admin

Web管理界面（无需认证）。

API统计
GET
/admin/api/stats

获取API调用统计数据。

重置统计
POST
/admin/api/clear

重置所有统计信息和历史记录。
